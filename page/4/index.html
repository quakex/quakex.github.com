再见，2014；你好，2015。

发表于 2014-12-31   |   分类于 日子   |  

###再见，2014。
2014 就这么稀里糊涂的过去了。我的目标似乎一件也没有完成。
想完成一个 APP，赚钱买个 MacBook。结果花钱买了一堆的书，却一本都没看完。
想维护一个高质量的 blog，出去找工作时有人要。结果一年写了2篇。还都是废话。
想把儿子培养得棒棒的，无论是身体上还是身心上。结果儿子几乎一个月病两遍，现在还在发烧。可怜的娃。
想精通 Rails 和 Mac、iOS 开发。结果专注入门20年。毛都没学成。
想多看书，提高点自己的修养。结果就看了一本小说《球状闪电》，还没看完。
写得心酸。好想找个豆腐撞死。
这一年，真的什么都没干。 一件有意义的事情都没做。

###2015就这么来了。

2015将是个挑战。有新生命的诞生。儿子也将进入中班。我也可能要做好走人的准备。困难重重，但我非常期待。
有挑战也好，2014太安逸了。仔细想想，留给我的时间真的不多了。不想再进入体制，几十岁的人了，要过点有尊严的生活，但还有家要养，所以，自己要有能力在离开体制后生存。还是有点难度的，竞争很激烈，毕竟岁数不小，虽然有点晚，但这个时代还是有机会的，只是要比别人付出多些。
不想多说，儿子还在发烧，我要早点休息好照顾他。

###2015 的目标：

  1. 完成一个 iOS、Mac APP 并上架。
  2. 坚持写 blog，书写是为了更好的思考。
  3. git everyday。
  4. 锻炼身体，减肥至130，锻炼肌肉、心肺功能。
  5. 当一个好爸爸、好丈夫、好儿子，控制自己的情绪，照顾好家人。
  6. 每天坚持学习，阅读。读书，做笔记。分享所学到的知识。
  7. 坚持英语学习，一年时间词汇量达到10000，学会基本的口语表达。

2015，加油，真的要加油，否则，真得去烤面筋了。

什么值得用之markdown

发表于 2013-06-09   |   分类于 tools   |  

markdown 是什么

我看书喜欢做笔记，为了区分章节、重点，专门设计一套标记规则。比如，按照字符在键盘上的次序，一级标题用@，二级用#，三级用%（为何不是#旁边的$，因为在中文下按$一般会显示￥，而%不会）。重点的内容就多打几个标记，比如两个@@。挺简洁的吧。:-)， 这套东西我还用得挺爽，直到我发现markdown~~~

我才明白为什么我一直不是一个好的程序员(还好编程只是我的业余爱好，否则我就饿死了)，因为相比较 markdown 我的那套东西是线性的，一维的，原始的。markdown则是多维的，是用程序员的角度设计的，是一门语言，就好比图灵用01解释了逻辑。markdown用很少的字符却实现了很多的功能，它足够简洁，你花十分钟就可掌握；又足够有效，写作的时候即完成排版；又足够美观，不像html，它通常只用一至两个字符即完成标记，完全不影响文章的阅读和美观。这点很重要。我们需要的就是这样的工具：简单、有效、美观。

我现在做笔记，写文章全部改用它了。一些笔记软件如果不支持它就果断放弃，比如Evernote，后面我还会写一篇文章说说我为什么弃 Evernote 而用 nvALT 以及我对软件的看法。总之自从用了markdown，从此腰不酸了，腿不疼了，一口气上五楼，连多年的老便秘都通了。

关于markdown标准介绍的文章有很多，我也懒得写了（文笔差，屎都快憋出来了才写这么点，唉，我语文难道是体育老师教的？）好吧，看这里，看这里 Markdown 语法说明，如果这篇不够直观，过于专业、刻板，我承认我也没看完。那么，请看这篇献给写作者的 Markdown 新手指南，很清楚了吧。有个地方没提，就是对程序员而言文章中怎么标记代码？很简单，代码前用Tab、四个空格、或缩进即可，给力吧。

看个实例吧：我做的《Pro Git》读书笔记。

语法介绍完了，介绍一下markdown的编辑器吧。

markdown 编辑器

国产神器：Mou

这个软件是由个人独立开发的，而且还是国人，非常不错。可免费使用，没有任何限制，业界良心啊。如果您觉得好用就给作者捐点款，个人开发者很辛苦，良好的开发生态环境需要大家一起来创造，中国程序员是被盗版给害苦了。

Mou支持各种快捷键，比如选中文字后，按cmd+b就是加粗，cmd+k就是内嵌代码，我一般用来标记关键词、中文里的英文等。cmd+i 就是斜体，cmd+u 就是删除线。cmd+[ cmd+] 就是左右缩进，选中多行文字后（比如大段代码）按这个快捷键就很方便。快捷键还有很多，慢慢尝试吧。总之，Mou是一款很用心的软件。但是也有一些缺点，我现在写blog用它，如果是做读书笔记，改用下面的这个大杀器了。

Sublime Text

不必多说，我现在用它。虽然没有mou好看，但是快，支持vim模式（emacs党滚粗），单词补全（这点很重要），装了插件后也可以预览、和快捷键编辑。

暂时写到这了，写东西真难啊，强烈谴责我所有的语文老师，怎么教的？
